/loop 自我迭代 — 无人监督持续执行
+0 min 新会话启动，读取 docs/task-queue.md 零上下文恢复状态
+1 min 找到可执行任务，委派 agent 或主线程执行 拓扑排序 + 并行
+3 min 闭环验证通过，commit，更新队列 标记 [x] + commit hash
+5 min 队列中还有 [ ] 任务 → 等待下一次 /loop 自然等待
... N 轮 全部 [x] → /loop 自然停止 完成
